Live eval

Cloxp got even more fun last week as it now has the ability to evaluate code as you type. Whereas recording values of expressions of runtime is a passive feature, live eval will drive evaluation of code changes. You can enable live evaluation in any clojure editor. Did I mention how much fun it is to work this way?

And yes, this feature is heavily inspired by LightTables instarepl.
